Weekly Rest & Events

Assigned Rest Days

In Deshangria, every worker is GUARANTEED rest. Each field of work has two assigned off-days per week. Workers whose last name begins with A–M take their rest on the earlier day of the cycle, while those with last names starting with N–Z rest the following day. This staggers the workforce so the city never sleeps all at once but everyon still gets a break. This is all happening not including if you had a weekend off so if you did lucky you!

The Rotating Breakfast Special

Every week, the Great Meal Hall cooks a new special breakfast. The day it's served shifts by one day later each week, so if this week it's on a Moonday, next week it'll be on a Townsday. This keeps things fresh and gives everyone in the city a chance to enjoy it on a day that suits them.

Ask the Help Center if you have any questions or concerns about this.
